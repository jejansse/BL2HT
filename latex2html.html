2.1 Answer Set Programming
Theorem

The immediate consequence operator of a program \(P\,\) is a \(\mathbb{B}_P \to [0,1]\) operator defined for an interpretation \(I\) of \(P\,\) and element \(a \in \mathbb{B}_P\) as: $$\Pi_{P}(I)(a) = \max \{ I(B(r)) \mid r \in P_a \}$$
